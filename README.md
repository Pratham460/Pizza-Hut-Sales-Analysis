# Pizza-Hut-Sales-Analysis

## Overview
This project analyzes Sales data for Pizza Hut using SQL to find trends, top customers, and revenue insights.

## Tools Used
- MySQL

## Dataset
- Pizza Hut Sales dataset (CSV)

## Key Analysis
- Top Pizzas by Revenue
- Category-wise distribution of Pizzas
- Distribution of Orders by time
